  1. <p> Tag

    • "align": Aligns the content of the paragraph.
    • "class": Sets one or more CSS classes to be applied to the paragraph.
    • "id": Defines a unique identifier for the paragraph.

  2. <img> Tag

    • "src": URL or path of the image to be displayed
    • "alt": Alternate text for when image cannot be displayed
    • "loading": Loading attribute on an <img> element specifies when to load the image. The 2 options are eager and lazy.

  3. <input> Tag

    • "type:" Specifies the type of input
    • "data-*": Defines additional data that can be used by JavaScript.
    • "hidden": Specifies whether the div element is hidden.

  4. <table> Tag

    • "border:" Sets the width of the border around the table and its cells.
    • "caption:" Adds a caption above or below the table.
    • "width:"" Defines the width of the table.

  • <form> Tag

    1. "action:" Specifies the URL where the form data will be submitted.
    2. "method:" Defines the HTTP method to be used for sending form data (e.g., "GET" or "POST").
    3. "enctype: Sets the encoding type for sending the form data to the server (e.g., "multipart/form-data").

  • <a> Tag

    1. "href:" Specifies the URL or destination the link points to
    2. "target:" Defines where to open the linked document (e.g., _blank for a new tab).
    3. "title:" Provides additional information about the link (shown as a tooltip).

  • <button> Tag

    1. "type:" Specifies the type of button (e.g., "button", "submit", "reset", etc.)
    2. "disabled: Disables the button, preventing user interaction.
    3. "form: Associates the button with a specific form element.

  • <ul> Tag

    1. "type:" Specifies the type of the list (e.g., "disc", "circle", or "square").
    2. "start:" Defines the starting number of the first list item.
    3. "compact:" Removes extra spacing between list items.
